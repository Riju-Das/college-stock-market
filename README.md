# Stock Market Management System

A simple MVP-style stock market management system with separate frontend and backend.

## Features
- User registration and login with access tokens
- View stock prices
- Buy and sell shares
- Portfolio summary and holdings
- Transaction history
- Market analysis snapshot

## Tech
- Backend: Node.js, Express, SQLite
- Frontend: React

## Running locally
1. Backend
   - cd backend
   - npm install
   - npm start
2. Frontend
   - cd frontend
   - npm install
   - npm run dev

The backend runs on port 4000 and the frontend runs on port 5173.

## Notes
- The SQLite database file is created at backend/data/market.db on first run.
